BIOMIO Captcha Implementation
  • 1. Copy and paste this code right before your </body> tag:
    <!-- jQuery is required, but if you already have it on your website, drop this line
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.9.1/jquery.min.js"></script>
-->
<!-- The main script -->
<script src="http://biom.io/biomio_captcha/main.js"></script>
  • 2. Copy and paste the code below into your page, wherever you would like the BIOMIO captcha to appear. 
    <div class="bio-captcha"></div>
  • 3. Add class .biomio-captcha-success to the submit button of the form where BIOMIO captcha is used
  • 4. Add the following function right before your on-submit function: 
    // this function will be triggered after successful captcha response
function bio-captcha-success() {
	//from here you can call your on-success function
}
