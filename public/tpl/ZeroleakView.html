How it works?

1. Sender attaches a file to the email.

2.Based on the settings and policies, the attached file is automatically converted to a special pdf format that requires authentication in order to open.

3.Recipient opens the file with the supplied file viewer User is authenticated using Biomio authenticator platform

4.File viewer restricts the ability to copy, take screenshots, take pictures, or show the document to non-authorized users.
